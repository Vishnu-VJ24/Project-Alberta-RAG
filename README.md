# Alberta Tourism Corridor RAG Project

This workspace supports an 8-week data science research project on tourism strategy for the Bow and Red Deer River corridors in Central Alberta.

Primary deliverables:

- Comprehensive tourism development report
- Strategic alignment analysis with Alberta Tourism Development Zones and sector strategy
- Transportation improvement plan
- Recreation adventure hub recommendations
- Marketing and branding strategy document
- RAG system that answers stakeholder questions using project evidence
